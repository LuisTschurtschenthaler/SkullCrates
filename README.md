# SkullCrates
Minecraft 1.12.2 Spigot-Plugin

Command | Description
------------ | -------------
crate list | List all crates
crate delete <crate> | Delete a crate
crate additems <crate> | Add all items from the inventory to the crate
crate create <name> <displayname> <skinURL> | Create a crate
crate giveall <crate> <amount> | Give everyone a crate
crate give <crate, *> <player> <amount> | Give one player a crates
